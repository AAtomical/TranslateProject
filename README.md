# TranslateProject

## 简介

HCTT 是“华科开放原子开源俱乐部”的翻译团队，参考“Linux 中国”翻译项目的运营模式，负责从国外优秀媒体翻译 Linux（特别是 Linux 内核）相关的技术、资讯、杂文等内容。
